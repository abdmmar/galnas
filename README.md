# galnas

Collection and API Documentation of Indonesian National Gallery. Built using Next.js

## Getting Started

First, run the development server:

```bash
# install dependecies
pnpm i

# run local server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Filter
  - Medium
  - Classifications
- Search
  - Title
- Sort
  - Title
  - Year
